# Task Manager (To-Do App)

## Overview
A professional JavaScript-based task management application designed to organize daily activities efficiently.

## Features
- Add, edit, and delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Task counter
- Input validation
- Clean and responsive UI

## JavaScript Concepts Used
- DOM manipulation
- Arrays and objects
- Event handling
- Conditional logic
- State management

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
