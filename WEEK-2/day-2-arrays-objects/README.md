# Student Marks List

## Project Purpose
This project demonstrates the use of JavaScript Arrays and Objects and displays student marks in a simple UI.

## Features
- Stores student data using objects inside an array
- Displays student marks in console
- Shows student data in a table
- Displays Pass/Fail status

## Technologies Used
- HTML
- CSS
- JavaScript

## Concepts Used
- Arrays
- Objects
- forEach loop
- DOM manipulation
