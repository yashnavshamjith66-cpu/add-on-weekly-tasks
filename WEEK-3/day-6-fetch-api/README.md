# Day 6 – Fetch API (Combined Tasks)

This task combines all Fetch API requirements into a single program.

## Tasks Covered

2. Convert response to JSON
3. Display required values with error handling

## Concepts Used

- async / await
- JSON parsing
- try / catch error handling
